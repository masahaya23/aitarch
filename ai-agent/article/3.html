ゲーム概要

平方完成をシューティングゲームで特訓できます。画面上部に ax²+bx の式が表示されるので、 落ちてくる5つの選択肢の中から正しく平方完成した式を撃ち落としてください。 係数ありの問題（例：3x²−9x）も出題され、分数を含む答えにも挑戦できます。 レベルが上がるほど落下速度が上がり、難易度が増します。

対象 高校数学（2次関数・平方完成）
問題数 無制限（ランダム生成）
形式 5択シューティング・3段階レベル
作成 HTML / CSS / JavaScript

遊び方

  • マウス移動または ←→ キーで砲台を左右に動かす
  • クリックまたはスペースキーでミサイルを発射する
  • 正解の式が書かれたブロックを撃ち落とすと得点
  • 誤答を撃つ、または正解が底まで落ちるとライフが減る
  • 5問ごとにレベルアップ（最大Level 3）
