議題・議事録作成アプリ

新しいタブで開きます

⚠️ ご利用には各自の Gemini API キーが必要です

本アプリは Google の Gemini API を利用します。共有のキーは用意されていません。 利用者ご自身で API キーを取得し、アプリの設定画面に登録してからご利用ください。

  • Google AI Studio で無料の API キー（AIza… で始まる文字列）を取得します
  • アプリの「⚙️ 設定」画面でキーを入力し、保存します
  • キーはお使いの端末（ブラウザ）内にのみ保存され、サーバーには送信・共有されません
  • キーは他人と共有せず、ご自身で管理してください

※ API の利用料金・利用枠は Google の規約に従います。登録するキーは各ユーザーの責任で管理してください。

アプリ概要

会議の「準備」から「議事録作成」までを AI で支援する、学校・職場向けの業務効率化ツールです。 議長は会議前に議題を整理してAIの改善提案や司会進行台本・タイムスケジュールを作成でき、 議事録係は会議後に文字起こしを貼り付けるだけで、議事録・要約・アクションプランを自動生成できます。 作成した内容は履歴として端末に保存され、いつでも見返せます。

対象 会議の議題準備・議事録作成を効率化したい教職員・社会人
特徴 議長モードと議事録係モードを搭載。文字起こしから議事録を自動生成
対応ファイル PDF・Word(.docx)・テキスト(.txt) の読み込みに対応
必要なもの 各ユーザーが取得・登録する Gemini API キー（上記参照）
形式 Webアプリ（ブラウザのみで動作、インストール不要）
開発者 藤井先生

2つのモード

🎯

議長モード（議長・主任向け）

議題メモを入力すると、AIが議題を改善し、司会進行台本やタイムスケジュールを作成します

📝

議事録係モード（議事録係向け）

文字起こしを貼り付けると、議事録・1分サマリー・アクションプランを自動生成します
