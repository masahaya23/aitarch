三角関数シューティング

ゲーム概要

弧度法による三角関数の値を答えるシューティングゲームです。 画面上から降ってくる5つの選択肢ブロックの中から正しい答えを砲台で撃ち落としてください。 sin → cos → tan の順に5問ずつ出題され、問題が進むほどブロックの落下速度が上昇します。

対象 高校数学（三角関数 / 数学II）
角度範囲 −2π ～ 2π（弧度法）
出題角度 π/6・π/4・π/3・π/2・π の整数倍
問題数 無制限（sin・cos・tan 各5問ループ）
作成 HTML / CSS / JavaScript（Canvas使用）

出題フェーズ

sin フェーズ

5問

答えの種類
0, ±1/2, ±√2/2, ±√3/2, ±1

cos フェーズ

5問

答えの種類
0, ±1/2, ±√2/2, ±√3/2, ±1

tan フェーズ

5問

答えの種類
0, ±1, ±√3, ±√3/3

操作方法

  • ← → カーソルキー または マウス移動：砲台を左右に動かす
  • スペースキー または クリック：ミサイルを発射
  • 正しい答えのブロックを撃つ → +10点
  • 誤ったブロックを撃つ、または正解ブロックを取り逃がす → ❤ 1減
  • 命（❤）が5つなくなるとゲームオーバー
